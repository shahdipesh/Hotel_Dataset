*********** The program runs on Mac computer ************

To Run the Program use the following command:
- make run

Note:
- The GUI might sometime not show up. In that case, just run the program again.
- When searching for all Hotels in a city, the program might take a while to show the results. This is because the program is searching for all hotels in the city and then filtering the results based on the user's input. This is done to ensure that the user gets the results.
